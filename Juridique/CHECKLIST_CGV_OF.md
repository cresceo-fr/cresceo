## 14. Synthèse

**Forces de la v1** :
- Format court et lisible, adapté à un démarrage
- Bonne spécificité sectorielle (Art. 5 EPI + Art. 7 floutage IA)
- Base solide sur annulation, prix, PI

**Lacunes critiques (à corriger pour v2)** :
1. Identification émetteur incomplète (adresse erronée, RCS, capital, NDA, Qualiopi)
2. RGPD trop succinct — non conforme (base légale, durée, droits, sous-traitants absents)
3. **Aucun droit de rétractation BtoC** → non-conforme code conso pour particuliers
4. **Médiateur consommation absent** → non-conforme code conso BtoC
5. **Indemnité 40€ recouvrement absente** → non-conforme L.441-10 BtoB
6. **Référent handicap absent** → risque audit Qualiopi
7. Politique de confidentialité référencée mais inexistante
8. Pas de distinction BtoB/BtoC claire
